Object of the game:

The object of the game is to make it across the highway without getting run over.
Make it to the top of the screen to advance to the next level. Each level is a little harder than the last, so be careful!

How to play:

Use the arrow keys to move the frog around the board. If the frog gets run over, it will be reset on the bottom sidewalk. Make it through all 20 levels to win the game!
